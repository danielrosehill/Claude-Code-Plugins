## Date Notation Conventions

The user likes dates to be written like this:

07-Nov-25

Where: 07 is the day of the month 
Nov is a shorthand for November 
25 is a shorthand for 2025

The user likes times to be written in 24 hour format always.

When planning schedules in other time zones, the user likes to note a UTC offset in addition to the local time.
